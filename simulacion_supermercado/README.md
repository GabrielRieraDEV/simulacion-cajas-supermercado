# Simulación de Cajas en Supermercados

Proyecto académico para modelar, simular y analizar diferentes configuraciones de cajas
(tradicionales, automáticas y rápidas) en el supermercado SuperLatino de Ciudad Bolívar.

## Estructura inicial

- `main.py`: punto de entrada del simulador.
- `config.py`: parámetros globales y configuraciones base.
- `simulation/`: lógica de simulación y procesos de colas.
- `analysis/`: utilidades para interpretación de resultados y generación de reportes.
- `ui/`: componentes de visualización (pygame/dash) y paneles de control.

> Esta versión solo establece la estructura para continuar el desarrollo en commits
> posteriores.
